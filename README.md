# Journey of the Heart — course website

Website for the NCKU micro-credit course **心路尋光：自我探索與溝通回應實務** (Journey of the Heart with Practical Self-Discovery and Communication Resilience), 114-2 semester.

Our teacher asked for a simple site to show what each group made in class, so here it is :)

🔗 **Live site:** https://pychs.github.io/JOURNEY-OF-THE-HEART-course-website/

## What's inside

- `index.html` — course intro, objectives, schedule and grading info (from the official syllabus)
- `group1.html` ~ `group6.html` — one page per group for their final projects
- `style.css` — shared styles for the whole site

## How it's built

Just plain HTML + CSS, no frameworks. Hosted for free on GitHub Pages, so any push to `main` updates the site automatically.
